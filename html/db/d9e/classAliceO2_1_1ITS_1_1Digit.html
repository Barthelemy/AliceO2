virtual bool AliceO2::ITS::Digit::operator< ( const Digit &  other) const
inlinevirtual

Test if the current digit is lower than the other Comparison is done based on the chip index and pixel index. Two options are possible for true:

  1. Chip index of this digit smaller than chip index of other digit
  2. Chip indices are equal, but pixel index of this chip is lower
    Parameters
    otherThe digit to compare with
    Returns
    True if this digit has a lower total index, false otherwise
